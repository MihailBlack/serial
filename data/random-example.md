# Сценарии кнопки "Кубики"

## Назначение
Кнопка "Кубики" показывает случайного участника из списка валидных анкет для быстрого нетворкинга.

## Алгоритм
1. Получить все анкеты.
2. Оставить только анкеты, где заполнены `fullName`, `phone`, `about`.
3. Если анкет 0 -> вернуть сообщение:
   - `Пока нет участников для знакомства`
4. Если анкета 1 -> вернуть эту анкету.
5. Если анкет 2+ -> выбрать случайный индекс:
   - `index = floor(random() * count)`
   - вернуть `profiles[index]`.

## Пример ответа при успешном выборе
```json
{
  "profile": {
    "id": "usr_012",
    "fullName": "Фролова Алина Константиновна",
    "phone": "+7 999 101-11-12",
    "about": "Занимаюсь HR-брендом и поиском специалистов в tech-команды."
  }
}
```

## Пример ответа, если показывать некого
```json
{
  "message": "Пока нет участников для знакомства"
}
```
